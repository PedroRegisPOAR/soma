1. Dados: $\mathrm{Q= {{cp.Q}} \ m^{3}/s}$ (vazão máxima), $\mathrm{T= {{17.5}} ^{\circ}C}$, $\mathrm{g= {{cp.g}} \ m/s^{2}}$.

De acordo com o trabalho de (PATTERSON; MORRIS, 1994) a desidade da água como uma função da temperatura é dada por:

$ \mathrm { \rho\left ( T \right )=a_{5}\left [ 1- \dfrac{\left ( T + a_{1} \right )^{2}\left ( T + a_{2} \right )}{a_{3}\left ( T + a_{4} \right )} \right ] } $


Conforme o capitulo 2, página 43 de (CENGEL; CIMBALA, 2012), a expressão para a viscosidade cinemática em função da temperatura é expressa por:

$ \mathrm { \mu\!\left ( T \right )=a10^{\dfrac{b}{T-c}} } $


Onde: $\mathrm{a=2.414 \!\cdot\!10^{-5} \ Ns/m^{2}, \ b=247.8 \ K, \ c=140 \ K}$.

Essa expressão conforme (CENGEL; CIMBALA, 2012) resulta em um erro menor do que 2.5% nos valores da viscosidade na faixa de temperatura de $0^{\circ}\mathrm{C}$ a $370^{\circ}\mathrm{C}$. Notar que a unidade de tempera das constantes é K (Kelvin). Convertendo para graus Celsius e substituindo as contantes:

$ \mathrm { \mu\!\left ( T \right )=2.414 \!\cdot\!10^{-5} \!\cdot\! 10^{\dfrac{247.8 }{T + 273.15 - 140}} \implies \mu\!\left ( T \right )=2.414 \!\cdot\!10^{-5} \!\cdot\! 10^{\dfrac{247.8 }{T + 133.15}} } $


Portanto essa é a formula para a viscosidade cinemática em função da temperatura.

2. Calculo da velocidade e da profundidade da água na seção de medição (seção 0):

$ \mathrm { H_{0}=kQ^{n} } \implies \mathrm { H_{0}= {{cp.k}} \cdot {{cp.Q}}^{ {{cp.N}} } } \implies \fbox { $\mathrm { H_{0}= {{cp.H0}} \ m }$ } $


$ \mathrm { D_{0}= \dfrac{2}{3} \left ( D - W \right ) + W } \implies \mathrm { D_{0}= \dfrac{2}{3} \left ( {{cp.D}} - {{cp.W}} \right ) + {{cp.W}} } \implies \fbox { $\mathrm { D_{0}= {{cp.D0}} \ m }$ } $


$ \mathrm { U_{0}= \dfrac{Q}{D_{0}H_{0}} } \implies \mathrm { U_{0}= \dfrac{ {{cp.Q}} }{ {{cp.D0}} \cdot {{cp.H0}} } } \implies \fbox { $\mathrm { U_{0}= {{cp.U0}} \ m/s }$ } $
3. Vazão especifica na garganta da Calha Parshall:

$ \mathrm { q= \dfrac{Q}{W} } \implies \mathrm { q= \dfrac{ {{cp.Q}} }{ {{cp.W}} } } \implies \fbox { $\mathrm { q= {{cp.q}} \ m\cdot s^{-1}/m }$ } $


4. Carga hidráulica disponível:

$ \mathrm { E_{0}= \dfrac{U_{0}^{2}}{2g} + H_{0} + N } \implies \mathrm { E_{0}= \dfrac{ {{cp.U0}}^{0}}{2\cdot {{cp.g}} } + {{cp.H0}} + {{cp.N}} } \implies \fbox { $\mathrm { E_{0}= {{cp.E0}} \ m } $ } $








5. Carga hidráulica disponível:

Seja: $ \mathrm{x=-gq\left ( \dfrac{2}{3} gE_{0}\right )^{-1.5}}$. Manipulando a expressão, tem-se:
$ \cos \! \left ( \theta \right ) =x \implies \theta=\arccos \! \left ( x \right ) \implies \dfrac{\theta}{3}=\ \dfrac{\arccos\left ( x \right )}{3} \implies \cos \! \left ( \dfrac{\theta}{3} \right ) = \cos \! \left ( \dfrac{\arccos \! \left( x \right )}{3} \right) $


Montando uma só expressão para a determinação da velocidade $\mathrm{U_{1}}$:

$ U_{1}=2\sqrt{\dfrac{2gE_{0}}{3}}\cos \! \left (\dfrac{1}{3}\arccos \! \left( -gq\left [\left ( \dfrac{2}{3} gE_{0}\right )^{-1.5} \right ] \right ) \right) $


Substituindo os valores:

$ U_{1}=2\sqrt{\dfrac{2 \cdot {{cp.g}} \cdot {{cp.E0}} }{3}}\cos \! \left (\dfrac{1}{3}\arccos \! \left( - {{cp.g}} \cdot {{cp.q}} \left [\left ( \dfrac{2}{3}\cdot {{cp.g}} \cdot {{cp.E0}} \right )^{-1.5} \right ] \right ) \right) $


$ \fbox { $ \mathrm{U_{1}}= {{cp.U1}} \mathrm{\ m/s} $ } $


Calculando $ \mathrm{h_{1}}$:

$ \mathrm { h_{1}= \dfrac{q}{U_{1}} } \implies \mathrm { h_{1}= \dfrac{ {{cp.q}} }{ {{cp.U1}} } } \implies \fbox{ $ \mathrm { h_{1}= {{cp.h1}} \ m } $ } $


6. Número de Froud:

$ \mathrm { F_{1}= \dfrac{U_{1}}{\sqrt{gh}} } \implies \mathrm { F_{1}= \dfrac{ {{cp.U1}} }{\sqrt{ {{cp.g}} \cdot {{cp.h}} }} } \implies \fbox { $\mathrm { F_{1}= {{cp.F1}} \ m }$ } $


7. Cálculo da altura do conjugada do ressalto (seção 2):

$ \mathrm { h_{2}= \dfrac{h_{1}}{2}\left ( \sqrt{1+ 8F_{1}^{2}} - 1\right ) } \implies \mathrm { h_{2}= \dfrac{ {{cp.h1}} }{2}\left ( \sqrt{1+ 8\cdot {{cp.F1}}^{2}} - 1\right ) } \implies \fbox { $\mathrm { h_{2}= {{cp.h2}} \ m }$ } $


8. Profundidade e velocidade d'água na seção de saída (seção 3):

$ \mathrm { h_{3}= h_{2} - \left ( N - K \right ) } \implies \mathrm { h_{3}= {{cp.h2}} - \left ( {{cp.N}} - {{cp.K}} \right ) } \implies \fbox { $\mathrm { h_{3}= {{cp.h3}} \ m }$ } $


$ \mathrm { U_{3}= \dfrac{Q}{Ch_{3}} } \implies \mathrm { U_{3}= \dfrac{ {{cp.Q}} }{ {{cp.C}} \cdot {{cp.h3}} } } \implies \fbox { $\mathrm { U_{3}= {{cp.U3}} \ m }$ } $


9. Extensão do ressalto e perda de carga:

$ \mathrm { L = 6\left ( h_{2} - h_{1} \right) } \implies \mathrm { L = 6\left ( {{cp.h2}} - {{cp.h1}} \right) } \implies \fbox { $\mathrm { L= {{cp.L}} \ m }$ } $


$ \mathrm { h = \dfrac{\left ( h_{2} - h_{1} \right)^{3}}{4h_{1}h_{2}} } \implies \mathrm { h = \dfrac{\left ( {{cp.h2}} - {{cp.h1}} \right)^{3} }{4\cdot {{cp.h1}} \cdot {{cp.h2}} } } \implies \fbox { $\mathrm { h= {{cp.h}} \ m }$ } $


10. Tempo de mistura e gradiente de velocidade:

$ \mathrm { T_{m} = \dfrac{2L}{\left ( U_{1} + U_{3} \right)} } \implies \mathrm { T_{m} = \dfrac{2\cdot {{cp.L}} }{\left ( {{cp.U1}} + {{cp.U3}} \right)} } \implies \fbox { $\mathrm { T_{m}= {{cp.Tm}} \ s }$ } $


$ \mathrm { G_{m} = \sqrt{\dfrac{\rho g h}{\mu T_{m}}} } \implies \mathrm { G_{m} = \sqrt{\dfrac{ {{cp.rho}} \cdot {{cp.g}} \cdot {{cp.h}} }{ {{cp.mu}} \cdot {{cp.Tm}} }} } \implies \fbox { $\mathrm { G_{m}= {{cp.Gm}} \ s^{-1} }$ } $


Como o valor do gradiente é maior do que $1000s^{-1}$, este valor pode ser usado.
