1. Dados: $\mathrm{Q= {{Q}} \ m^{3}/s}$, $\mathrm{T= {{T}} ^{\circ}C}$, $\mathrm{g= {{g}} \ m/s^{2}}$.

Escolehndo a calha parshal com as seguintes dimensões:

$\mathrm{ W = {{W}} \ m}$, $\mathrm{ C = {{C}} \ m}$, $\mathrm{ D = {{D}} \ m}$, $\mathrm{ N = {{N}} \ m}$, $\mathrm{ k = {{k}} }$, $\mathrm{ n = {{n}} \ m}$

De acordo com o trabalho de (PATTERSON; MORRIS, 1994) a desidade da água como uma função da temperatura é dada por:

$\mathrm { \rho \!\left ( T \right )= \rho_{0} \! \Big\{1- \Big[ A\!\left (t - t_{0} \right ) + B\!\left (t - t_{0} \right )^{2} + C\!\left (t - t_{0} \right )^{3} + D\!\left (t - t_{0} \right )^{4} + E\! \left (t - t_{0} \right ) ^{5} \Big] \Big\} }$


Onde: $\mathrm { A = 7.0134 \times 10^{-8 \ \circ}C^{-1}, B = 7.926504 \times 10^{-6 \ \circ}C^{-2}, C = -7.575677 \times 10^{-8 \ \circ}C^{-3}, \\ D = 7.314894 \times 10^{-10 \ \circ}C^{-4}, E = -3.596458 \times 10^{-12 \ \circ}C^{-5}, t_{0}=3.9818^{\circ}C, \rho= 999.974950 Kg/m^{3} }$

Esta formula segundo os seus autores tem um erro de 0.64 partes em $10^{6}$ para $15^{\circ}\mathrm{C}$, 1.4 partes em $10^{6}$ para $1^{\circ}\mathrm{C}$, e para $40^{\circ}\mathrm{C}$

Substituindo-se obtem-se: $\mathrm { \fbox { $\mathrm { \rho= {{rho}} \ Kg/m^{3} }$ } } $


Conforme o capitulo 2, página 43 de (CENGEL; CIMBALA, 2012), a expressão para a viscosidade cinemática em função da temperatura é expressa por:

$ \mathrm { \mu\!\left ( T \right )=a10^{\dfrac{b}{T-c}} } $


Onde: $\mathrm{a=2.414 \!\cdot\!10^{-5} \ Ns/m^{2}, \ b=247.8 \ K, \ c=140 \ K}$.

Essa expressão conforme (CENGEL; CIMBALA, 2012) resulta em um erro menor do que 2.5% nos valores da viscosidade na faixa de temperatura de $0^{\circ}\mathrm{C}$ a $370^{\circ}\mathrm{C}$. Notar que a unidade de tempera das constantes é K (Kelvin). Convertendo para graus Celsius e substituindo as contantes:

$ \mathrm { \mu\!\left ( T \right )=2.414 \!\cdot\!10^{-5} \!\cdot\! 10^{\dfrac{247.8 }{T + 273.15 - 140}} \implies \mu\!\left ( T \right )=2.414 \!\cdot\!10^{-5} \!\cdot\! 10^{\dfrac{247.8 }{T + 133.15}} } $


Portanto essa é a formula para a viscosidade cinemática em função da temperatura.

$ \mathrm { \mu\!\left ( T \right )=2.414 \!\cdot\!10^{-5} \!\cdot\! 10^{\dfrac{247.8 }{T + 133.15}} } \implies \mathrm { \mu\!\left ( {{T}} \right )=2.414 \!\cdot\!10^{-5} \!\cdot\! 10^{\dfrac{247.8 }{ {{T}} + 133.15}} } \\ \implies \fbox { $\mathrm { \mu= {{mu}} \ N \! \cdot \! s \cdot m^{-2} }$ } $












2. Calculo da velocidade e da profundidade da água na seção de medição (seção 0):

$ \mathrm { H_{0}=kQ^{n} } \implies \mathrm { H_{0}= {{k}} \cdot {{Q}}^{ {{N}} } } \implies \fbox { $\mathrm { H_{0}= {{H0}} \ m }$ } $


$ \mathrm { D_{0}= \dfrac{2}{3} \left ( D - W \right ) + W } \implies \mathrm { D_{0}= \dfrac{2}{3} \left ( {{D}} - {{W}} \right ) + {{W}} } \implies \fbox { $\mathrm { D_{0}= {{D0}} \ m }$ } $


$ \mathrm { U_{0}= \dfrac{Q}{D_{0}H_{0}} } \implies \mathrm { U_{0}= \dfrac{ {{Q}} }{ {{D0}} \cdot {{H0}} } } \implies \fbox { $\mathrm { U_{0}= {{U0}} \ m/s }$ } $


3. Vazão especifica na garganta da Calha Parshall:

$ \mathrm { q= \dfrac{Q}{W} } \implies \mathrm { q= \dfrac{ {{Q}} }{ {{W}} } } \implies \fbox { $\mathrm { q= {{q}} \ m^{3}\cdot s^{-1}/m }$ } $


4. Carga hidráulica disponível:

$ \mathrm { E_{0}= \dfrac{U_{0}^{2}}{2g} + H_{0} + N } \implies \mathrm { E_{0}= \dfrac{ {{U0}}^{2}}{2\cdot {{g}} } + {{H0}} + {{N}} } \implies \fbox { $\mathrm { E_{0}= {{E0}} \ m } $ } $


5. Carga hidráulica disponível:

Seja: $ \mathrm{x=-gq\left ( \dfrac{2}{3} gE_{0}\right )^{-1.5}}$. Manipulando a expressão, tem-se:
$ \cos \! \left ( \theta \right ) =x \implies \theta=\arccos \! \left ( x \right ) \implies \dfrac{\theta}{3}=\ \dfrac{\arccos\left ( x \right )}{3} \implies \cos \! \left ( \dfrac{\theta}{3} \right ) = \cos \! \left ( \dfrac{\arccos \! \left( x \right )}{3} \right) $


Montando uma só expressão para a determinação da velocidade $\mathrm{U_{1}}$:

$ U_{1}=2\sqrt{\dfrac{2gE_{0}}{3}}\cos \! \left (\dfrac{1}{3}\arccos \! \left( -gq\left [\left ( \dfrac{2}{3} gE_{0}\right )^{-1.5} \right ] \right ) \right) $


Substituindo os valores:

$ U_{1}=2\sqrt{\dfrac{2 \cdot {{g}} \cdot {{E0}} }{3}}\cos \! \left (\dfrac{1}{3}\arccos \! \left( - {{g}} \cdot {{q}} \left [\left ( \dfrac{2}{3}\cdot {{g}} \cdot {{E0}} \right )^{-1.5} \right ] \right ) \right) $


$ \fbox { $ \mathrm{U_{1}}= {{U1}} \mathrm{\ m/s} $ } $


Calculando $ \mathrm{h_{1}}$:

$ \mathrm { h_{1}= \dfrac{q}{U_{1}} } \implies \mathrm { h_{1}= \dfrac{ {{q}} }{ {{U1}} } } \implies \fbox{ $ \mathrm { h_{1}= {{h1}} \ m } $ } $




6. Número de Froud:

$ \mathrm { F_{1}= \dfrac{U_{1}}{\sqrt{gh}} } \implies \mathrm { F_{1}= \dfrac{ {{U1}} }{\sqrt{ {{g}} \cdot {{h}} }} } \implies \fbox { $\mathrm { F_{1}= {{F1}} }$ } $


7. Cálculo da altura do conjugada do ressalto (seção 2):

$ \mathrm { h_{2}= \dfrac{h_{1}}{2}\left ( \sqrt{1+ 8F_{1}^{2}} - 1\right ) } \implies \mathrm { h_{2}= \dfrac{ {{h1}} }{2}\left ( \sqrt{1+ 8\cdot {{F1}} ^{2}} - 1\right ) } \implies \fbox { $\mathrm { h_{2}= {{h2}} \ m }$ } $


8. Profundidade e velocidade d'água na seção de saída (seção 3):

$ \mathrm { h_{3}= h_{2} - \left ( N - K \right ) } \implies \mathrm { h_{3}= {{h2}} - \left ( {{N}} - {{K}} \right ) } \implies \fbox { $\mathrm { h_{3}= {{h3}} \ m }$ } $


$ \mathrm { U_{3}= \dfrac{Q}{Ch_{3}} } \implies \mathrm { U_{3}= \dfrac{ {{Q}} }{ {{C}} \cdot {{h3}} } } \implies \fbox { $\mathrm { U_{3}= {{U3}} \ m }$ } $


9. Extensão do ressalto e perda de carga:

$ \mathrm { L = 6\left ( h_{2} - h_{1} \right) } \implies \mathrm { L = 6\left ( {{h2}} - {{h1}} \right) } \implies \fbox { $\mathrm { L= {{L}} \ m }$ } $


$ \mathrm { h = \dfrac{\left ( h_{2} - h_{1} \right)^{3}}{4h_{1}h_{2}} } \implies \mathrm { h = \dfrac{\left ( {{h2}} - {{h1}} \right)^{3} }{4\cdot {{h1}} \cdot {{h2}} } } \implies \fbox { $\mathrm { h= {{h}} \ m }$ } $


10. Tempo de mistura e gradiente de velocidade:

$ \mathrm { T_{m} = \dfrac{2L}{\left ( U_{1} + U_{3} \right)} } \implies \mathrm { T_{m} = \dfrac{2\cdot {{L}} }{\left ( {{U1}} + {{U3}} \right)} } \implies \fbox { $\mathrm { T_{m}= {{Tm}} \ s }$ } $


$ \mathrm { G_{m} = \sqrt{\dfrac{\rho g h}{\mu T_{m}}} } \implies \mathrm { G_{m} = \sqrt{\dfrac{ {{rho}} \cdot {{g}} \cdot {{h}} }{ {{mu}} \cdot {{Tm}} }} } \implies \fbox { $\mathrm { G_{m}= {{Gm}} \ s^{-1} }$ } $


Como o valor do gradiente é maior do que $1000s^{-1}$, este valor pode ser usado. {% endblock %}
