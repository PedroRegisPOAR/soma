Calculos do modelo de projeção com taxa decrescente de crescimento:

$ \mathrm { P_{s} = \dfrac{ 2P_{0}P_{1}P_{2} - P_{1}^{2}\left ( P_{0} + P_{2} \right ) }{ P_{0}P_{2} - P_{1}^{2} } } \\ \implies \mathrm { P_{s} = \dfrac{ 2\cdot {{P0}}\cdot {{P1}}\cdot {{P2}} - {{P1}}^{2}\left ( {{P0}} + {{P2}} \right ) }{ {{P0}}\cdot {{P2}} - {{P1}}^{2} } } \\ \implies \fbox { $\mathrm { P_{s} = {{Ps}} }$ } $


$ \mathrm { K_{1} = \dfrac{1}{t_{2} - t_{1} } \ln \!\!\left [ \dfrac{ P_{0}\left (P_{s} - P_{1} \right ) }{P_{1}\left (P_{s} - P_{0} \right )} \right ] } \\ \implies \mathrm { K_{1} = \dfrac{1}{ {{t2}} - {{t1}} } \ln \!\!\left [ \dfrac{ {{P0}}\left ( {{Ps}} - {{P1}} \right ) }{ {{P1}} \left ( {{Ps}} - {{P0}} \right )} \right ] } \\ \implies \fbox { $\mathrm { K_{1} = {{K1}} }$ } $


$ \mathrm { c = \dfrac{ P_{s} - P_{0} }{ P_{0} } } \implies \mathrm { c = \dfrac{ {{Ps}} - {{P0}} }{ {{P0}} } } \implies \fbox { $\mathrm { c = {{c}} }$ } $

$ \mathrm { P\left ( t \right ) = \dfrac{ P_{s} }{ 1 + c\cdot e^{ K_{1}\left ( t - t_{0} \right )} } } \implies \fbox { $\mathrm { P\left ( t \right ) = \dfrac{ {{Ps}} }{ 1 + {{c}}\cdot e^{ {{K1}}\left ( t - {{t0}} \right )} } }$ } $
