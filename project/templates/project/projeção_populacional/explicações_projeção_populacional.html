Algumas explicações sobre o funcionamento do sistema da calculo.

Formatos dos gráficos Graficos não presentes Erros Vocês, usuario pode cometer alguns erros bem estranhos, então aqui eu tentei documentar alguns erros que o sistema tenta prevenir.

Caso sejam colocados tempo iguais o sistema não aceitará.

O calculo das projeções esta feito para que sejam apenas exibidas as projeções que o sistema conseguiu realizar os calculos.

Acredito, pois é o que todos os testes que eu fiz mostram, que as projeções aritimética (PA) e geométrica (PG) possam ser calculadas para "quais quer dados de entrada". Então a menos as projeções de PA e PG devem ser possiveis de ser feitas.

Para a Taxa Decrescente (TD):

$ \mathrm { P_{s} = \dfrac{ 2P_{0}P_{1}P_{2} - P_{1}^{2}\left ( P_{0} + P_{2} \right ) }{ P_{0}P_{2} - P_{1}^{2} } } $


Observem que a condição de existência da expressão para o $\mathrm{P_{s}}$ é que o denominador seja diferente de zero, ou seja, $\mathrm{P_{0}P_{2} - P_{1}^{2}\neq 0}$.

Ainda tem-se a condição de existenicia da expressão do $\mathrm{K_{1}}$:
$ \mathrm { K_{1} = \dfrac{1}{t_{2} - t_{1} } \ln \!\!\left [ \dfrac{ P_{0}\left (P_{s} - P_{1} \right ) }{P_{1}\left (P_{s} - P_{0} \right )} \right ] } $


Lembre-se de que o dominio da função logaritmica é o conjunto dos numeros rais positivos. Então para alguns valores de $\mathrm{P_{0}, P_{1}, P_{2} }$, é possivel que o argumento do logaritmo assuma voleres negativos, valores estes que não fazem parte do dominio da função logaritmica.

Para a o Crescimento Logistico (CL):

Semelhante ao que foi discutido no TD o $\mathrm{P_{s}}$ não pode ser calculado para qualquer valor de $\mathrm{P_{0}, P_{1}, P_{2} }$.

$ \mathrm { P_{s} = \dfrac{ 2P_{0}P_{1}P_{2} - P_{1}^{2}\left ( P_{0} + P_{2} \right ) }{ P_{0}P_{2} - P_{1}^{2} } } $


Para a condição de existência passada na imagem pelo professor Hélio é utilizada a condição $\mathrm{P_{0}P_{2} < P_{1}^{2}}$, que atende que o denominador seja diferente de zero, pois $\mathrm{P_{0}P_{2} < P_{1}^{2} \implies P_{0}P_{2} - P_{1}^{2} < 0}$.

Para o CL é utilizada tambem a condição de que $\mathrm{P_{0} < P_{1} < P_{2} }$.

Caso mesmo com essas condições ocorra algum problema não será exibido essa projeção.





Uma pergunta que me ocorreu foi. Existem $\mathrm{P_{0}, P_{1}, P_{2} }$ tais que que não seja possivel calcular fazer a projação que usa o modelo com taxa decrescente, mas é possivel fazer a projeção do modelo logistico?

A conclusão que cheguei é que não

Essa impossibilidade advem da condição de existencia dos logaritmos presentes para o calculo das constantes $\mathrm{K_{1} }$ e $\mathrm{K_{d} }$.
$ \mathrm { P_{s} = \dfrac{ 2P_{0}P_{1}P_{2} - P_{1}^{2}\left ( P_{0} + P_{2} \right ) }{ P_{0}P_{2} - P_{1}^{2} } } $


$ \mathrm { K_{1} = \dfrac{1}{t_{2} - t_{1} } \ln \!\!\left [ \dfrac{ P_{0}\left (P_{s} - P_{1} \right ) }{P_{1}\left (P_{s} - P_{0} \right )} \right ] } $


$ \mathrm { K_{d} = \dfrac{ \ln \!\!\left ( \dfrac{P_{s} - P_{2}}{P_{s} - P_{0}} \right )}{t_{2} - t_{1} } } $


Pala condição de existência dos logaritmos tem-se qu se ter:

$ \mathrm { \dfrac{ P_{0}\left (P_{s} - P_{1} \right ) }{P_{1}\left (P_{s} - P_{0} \right )} > 0 } $ e $ \mathrm { \dfrac{P_{s} - P_{2} }{P_{s} - P_{0} } > 0 } $


Prova.

Fazendo: $ \mathrm { M = 2P_{0}P_{1}P_{2} - P_{1}^{2}\left ( P_{0} + P_{2} \right ) } $ e $ \mathrm { N = P_{0}P_{2} - P_{1}^{2} } \implies \mathrm { P_{s} = \dfrac{M}{N} } $

Substituindo nas expressões, e simplificando:

$ \mathrm { \dfrac{P_{s} - P_{2} }{P_{s} - P_{0} } } \implies \mathrm { \dfrac{\dfrac{M}{N} - P_{2} }{\dfrac{M}{N} - P_{0} } } \implies \mathrm { \dfrac{ \left (\dfrac{M - NP_{2}}{N} \right ) }{ \left ( \dfrac{M - NP_{0}}{N} \right )} } \implies \mathrm { \dfrac{ \dfrac{\left (M - NP_{2} \right )}{N} }{\dfrac{ \left (M - NP_{0}\right )}{N} } } \implies \mathrm { \dfrac{ M - NP_{2} }{ M - NP_{0} } } $


$ \mathrm { \dfrac{ P_{0}\left (\dfrac{M}{N} - P_{1} \right ) }{P_{1}\left ( \dfrac{M}{N} - P_{0} \right )} } \implies \mathrm { \dfrac{ P_{0}\left (\dfrac{M - NP_{1}}{N} \right ) }{P_{1}\left ( \dfrac{M - NP_{0}}{N} \right )} } \implies \mathrm { \dfrac{ \dfrac{P_{0}\left (M - NP_{1} \right )}{N} }{\dfrac{P_{1} \left (M - NP_{0}\right )}{N} } } \implies \mathrm { \dfrac{ P_{0}\left (M - NP_{1} \right ) }{ P_{1} \left (M - NP_{0}\right ) } } $


O problema agora consiste em mostrar que se (IV) é positiva, (III) tambem é. Para isso basta lembrar que oara realizar a projeção populacional que utiliza o método do crecimento logistico é necessario que $\mathrm{ P_{0}P_{2} < P_{1}^{2} }$, ou seja, que $\mathrm{ P_{0}P_{2} - P_{1}^{2} < 0 }$.

Se $\mathrm{ P_{0}P_{2} - P_{1}^{2} < 0 \implies N < 0 }$. Logo:

$ \mathrm { \dfrac{ P_{0}\left (M - NP_{1} \right ) }{ P_{1} \left (M - NP_{0}\right ) } } \implies \mathrm { \dfrac{ P_{0}\left (M - \left ( - N\right )P_{1} \right ) }{ P_{1} \left (M - \left ( - N\right )P_{0}\right ) } } \implies \mathrm { \dfrac{ P_{0}\left (M + NP_{1} \right ) }{ P_{1} \left (M + NP_{0}\right ) } } $


$ \mathrm { \dfrac{ P_{0}\left (M - NP_{1} \right ) }{ P_{1} \left (M - NP_{0}\right ) } } \implies \mathrm { \dfrac{ P_{0}\left (M - \left ( - N\right )P_{1} \right ) }{ P_{1} \left (M - \left ( - N\right )P_{0}\right ) } } \implies \mathrm { \dfrac{ P_{0}\left (M + NP_{1} \right ) }{ P_{1} \left (M + NP_{0}\right ) } } $


Ou seja, se a condição para o argumento do logaritmo da concstante do crecimento logistico é satisfeita Isso conclui a demonstração.
