Calculos do modelo de projeção com taxa decrescente de crescimento:

$ \mathrm { P_{s} = \dfrac{ 2P_{0}P_{1}P_{2} - P_{1}^{2}\left ( P_{0} + P_{2} \right ) }{ P_{0}P_{2} - P_{1}^{2} } } \\ \implies \mathrm { P_{s} = \dfrac{ 2\cdot {{P0}}\cdot {{P1}}\cdot {{P2}} - {{P1}}^{2}\left ( {{P0}} + {{P2}} \right ) }{ {{P0}}\cdot {{P2}} - {{P1}}^{2} } } \\ \implies \fbox { $\mathrm { P_{s} = {{Ps}} }$ } $


$ \mathrm { K_{d} = \dfrac{ \ln \!\!\left ( \dfrac{P_{s} - P_{2}}{P_{s} - P_{0}} \right )}{t_{2} - t_{1} } } \implies \mathrm { K_{d} = \dfrac{ \ln \!\!\left ( \dfrac{ {{Ps}} - {{P2}} }{ {{Ps}} - {{P0}} } \right ) }{ {{t2}} - {{t1}} } } \implies \fbox { $\mathrm { K_{d} = {{Kd}} }$ } $


Substituindo-se os valores formula tem-se:

$ \mathrm { P\left ( t \right ) = P_{0} + \left ( P_{s} - P_{0} \right )\left ( 1 - e^{-K_{d}\left ( t - t_{0} \right )} \right ) } \implies \fbox { $\mathrm { P\left ( t \right ) = {{P0}} + \left ( {{Ps}} - {{P0}} \right )\left ( 1 - e^{-{{Kd}}\left ( t - {{t0}} \right )} \right ) }$ } $
