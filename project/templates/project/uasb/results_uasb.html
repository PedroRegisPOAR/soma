9) Verificação das cargas aplicadas.

Carga orgânica volumétrica:

A formula geral para a carga orgânica volumétrica é: $\mathrm{ COV= \dfrac{S_{0}Q}{V} }$, e para o UASB é usada como: $\mathrm{ COV= \dfrac{ DQO_{af} Q_{Méd} }{ V_{TotCorr}} }$

$ \mathrm { COV= \dfrac{ DQO_{af} Q_{Méd} }{ V_{TotCorr}} } \implies \mathrm { COV= \dfrac{ {{DQOaf}} \frac{g}{L} \, \frac{1Kg}{1000g} \, \frac{1000L}{1m^{3}}\ {{Qméd}} \frac{m^{3}}{dia} }{ {{VTotCorr}} \, m^{3} } } \implies \fbox { $\mathrm { COV = {{COV}} \ \dfrac{KgDQO}{m^{3} dia} }$ } $


O valor do COV é menor do que o $\mathrm{ COV_{Máx} }$ que é {{COVMáx}} $\mathrm{ \frac{KgDQO}{m^{3}dia} }$.

Carga hidráulica volumétrica:

A formula geral para a carga hidraulica volumétrica é: $\mathrm{ CHV= \dfrac{Q}{V} }$, e para o UASB é usada como: $\mathrm{ COV= \dfrac{ Q_{Méd} }{ V_{TotCorr}} }$

$ \mathrm { CHV= \dfrac{ Q_{Méd} }{ V_{TotCorr}} } \implies \mathrm { CHV= \dfrac{ {{Qméd}} m^{3}/dia }{ {{VTotCorr}} m^{3}} } \implies \fbox { $\mathrm { CHV = {{CHV}} \ \dfrac{m^{3}}{m^{3} dia} }$ } $


O valor da CHV é menor do que o $\mathrm{ CHV_{Máx} }$ que é {{CHVMáx}} $\mathrm{ \frac{m^{3}}{m^{3}dia} }$.
